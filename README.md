# astrbot-plugin-cet6
# 🎓 AstrBot 插件：四六级金牌私教 (CET-6 Tutor)

这是一个专为 AstrBot 打造的沉浸式英语四六级备考插件！内置经过精工排版的真题阅读库与核心词汇库。不仅能刷题，还能当场出分、查单词。

## ✨ 核心功能
* **`/来篇阅读`**：随机抽取一篇极其干净、无错版的六级阅读真题（支持三重精准匹配防乱码）。
* **`/答案 ABCD`**：做完瞬间批改，输出可视化成绩单。
* **`/抽单词`**：从核心词库中随机抽出 5 个乱序单词进行闪测。
* **`/查单词 [英文]`**：一键查询单词的详细释义和例句。

## 📦 安装说明
1. 在 AstrBot 管理面板的“插件市场”搜索 `astrbot_plugin_cet6` 一键安装。
2. 或直接下载本仓库 ZIP 包，上传至 AstrBot 后台。
3. **注意**：本插件自带极其珍贵的清洗版 JSON 题库，安装后无需额外配置即可满血运行！
